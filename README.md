# PBSS (Pitt Badminton Sign-up System)
A program that replaces the sign-up sheet of Pitt's badminton club

# How to use
(Assuming you have java installed)
1. Download PBSS.java and PBSSdriver.java
2. Type the following command into your command line program:
```
javac PBSS.java
java PBSSdriver.java
```
3. voilà
